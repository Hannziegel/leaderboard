# Leaderboard:
- A Website application to show and add scores to a leaderboard made with JavaScript, Webpack, and leaderboard API.

## Acces link

[Live Demo](https://hannziegel.github.io/leaderboard/dist/)

## Setup

      Clone this repo

      cd Leaderboard

      npm install

      npm start
